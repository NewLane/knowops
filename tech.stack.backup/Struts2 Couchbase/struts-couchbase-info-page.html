Struts2 with NoSQL

Intent

To create an application that uses:
  • Struts2 MVC Framework
  • JQuery/Boostrap Javascript front end framework
  • Spy Memcached for POJO caching
  • Couchbase for NoSQL document level persistence
  • JUnit for testing CRUD capabilities
  • Maven for app building, packaging, and deployment

Key Maven Elements

  • Struts2 v2.3.8
  • Spy Memcached v2.12.0
  • Couchbase Java Client v2.3.6
  • Jetty
  • Tomcat Jasper
  • JUnit

Dependencies

The following must be installed:
  • Java 8 - download
  • Maven 3.x - download
  • Couchbase Server 3.x - download

Special Notes

  • Uses the Couchbase client to interact with Couchbase as a Java client.
  • Run a created app in the Maven Jetty container using the following command: mvn jetty:run
  • To deploy a generated app to a different web server, build its WAR file using the following command: mvn clean install
  • Run the Couchbase server to create a document bucket. Then provide the bucket name and other Couchbase options in the Options screen
  • Javascript and CSS3 are provided through Bootstrap 3.3.7 and JQuery 3.1.1
