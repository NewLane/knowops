Today, we want to help you get familiar with realMethods.

There are 3 ways to use realMethods and we suggest first exploring 1 or 2:

  1. Using the CLI (Command Line Interface)
    The CLI gives you full access to all the features of the realMethods platform.

  2. Using the Web Client
    The Web Client offers a simple to use UI to generate an application in a few easy steps.

  3. Using the API
    The API offers the most control for scenarios where you want to programmatically include application generation into your application delivery process.

Are you a CircleCI user?

realMethods has created a CircleCI Orb to help you generate applications which are in turn deployed directly to your instance on CircleCI. There is no faster way to get DevOps up and running.
